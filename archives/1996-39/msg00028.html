I just learned of a lyrics server at this URL:

http://archive.uwp.edu/pub/web/lyrics/

I have been a Kate fan since The Kick Inside but honestly just never
listened to lyrics that closely. So with all this discussion about the
meaning of The Ninth Wave leaving me behind, I looked up the lyrics
and printed them out so I can read them while listening to the CD
and see what interpretations I come up with.

This server is an invaluable resource as it has thousands of artists
listed. But there is a warning that copyright lawyers are trying to 
shut down the page so you may want to get what you want while the 
getting is good.
