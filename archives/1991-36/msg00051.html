All right, differences:

1) Japanese set uses Japanese cover for The Kick Inside, and different
cover for Never For Ever (close-up of regular cover).  British set
uses British covers.

2) Japanese set includes ALL the lyrics, including lyrics for the two
CDs of B-sides, etc.  British set doesn't include lyrics for those two
extra CDs.

3) More biographical information in the Japanese set, but it's all
in Japanese.

4)  Someone told me that the sound quality is better in the Japanese
set, but I don't know for a fact if that's true.

5) No stickers in the Japanese set.
