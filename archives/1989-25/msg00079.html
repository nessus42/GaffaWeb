After finally fixing the news server server inews backend here
(sigh), I think this message should finally get through.

I converted the above image to a Sun 1 bit raster file using the fbm
package.  This file is displayable under SunView or X11.  To view
under X11, you need to get the xbgsun program from your nearby X
archive.  Sun rasterfiles display _much_ faster than X11 bitmaps, so
this is a program to have...

Pick up the file pub/tsw.sun.Z (remember to use binary mode) using
anonymous ftp to ics.uci.edu (128.195.1.1).

Enjoy!
