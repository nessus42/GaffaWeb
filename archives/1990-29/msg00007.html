Subject: This Woman's Work Box Set

One thing I noticed about the Box Set: the first five albums have higher
quality paper in the original UK CD release.  The paper used in the
booklets included with the Box Set is thinner.  _The Kick Inside_,
_Lionheart_, and _Never for Ever_ all have thicker paper for the covers
of the booklets, and _The Dreaming_ and _Hounds of Love_ use thicker
paper for the entire booklet.  Otherwise, the differences are minor.
The colours on the _Never for Ever_ booklet seem brighter on the Box Set
version, but maybe it's because my original _NFE_ booklet has faded.

Sure is nice having the rarities CDs though.
