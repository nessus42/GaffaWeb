While home in Vancouver for American Thanksgiving, I found the CD boxed
set on sale for $83.94 CDN + tax ($75 US).  This was at the downtown
A&B Sound (Seymour St.)  Unfortunately, I got the last CD set in stock;
there were several cassette versions still available and A&B has four
other locations in the area that might have the CDs.  So with any
luck, it is possible to get the set almost half price.  I think A&B's
list price is $139.99 CDN.  Down the street at A&A, the CD boxed set was
available for $100 CDN.

Hope this helps,
