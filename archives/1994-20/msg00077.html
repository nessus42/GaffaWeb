Chris here,

    I have uploaded the first Kate audio MPEG to /tmp/gaffa on
ftp.uu.net. MPEG (level 3) is an amazing format and is the only
format to produce any meaningful compression. The uncompressed
version of this file is 19,477,193 bytes. The MPEG compressed
version is only 1,762,080 bytes. MPEG is the defacto standard
for Mosiac browsers, but if you don't have a an MPEG player
yet, I have included the MS-DOS executables as level3.zip.
l3dec_fp.exe can convert the MPEG back to a .WAV file. *Please*
don't convert this to the awful 8-bit, 11khz Sun AU format, but
if you do, please don't upload it anywhere.

    This is a very good quality version of the song "Another Day"
from the Christmas Special _Kate_. It has never been officially
released on *any* medium, and features Kate and Peter Gabriel
singing a song by Roy Harper. 

    The recording was done at 44khz sampling rate at 16 bits on
a Sound Blaster 16. I'd like to know how it works on other
platforms.

    MPEG is the wave of the future. Please check it out.
