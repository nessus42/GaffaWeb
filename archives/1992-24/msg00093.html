More info about the NEVER FOR EVER box:

The release date is supposedly set for September 22. The set will
contain a T-shirt and a photo booklet, and is limited to 1,500
copies worldwide. Intergalactic garage is advertising it preordered
for $69.95 + 5 bucks postage. ($75 plus postage after release).
I assume that C-Side records will carry this piece as well.

Is there currently an interview album discography on Kate anywhere?
Although such a task would be rather difficult, it would prevent
some of us from buying the same interview twice (as so many of
these releases continually recycle the same old interviews in new
packages).
