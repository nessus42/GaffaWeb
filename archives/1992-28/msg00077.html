Hello to all Harmonica lovers,

I have just had a list server for harmonica created.  This list is
open to anyone who plays any type of music, who likes any type of
music and those who are just curious.  EVERYONE is WELCOME!!!

This is for no certain type of music.  Personally, I would like to see
a large amount of diversity.  Blues, Jazz, Folk, Classical, Country,
Alternative, Industrial, Punk and all other styles of harmonica playing
are welcome.

To subscribe to this server write (via e-mail) to:

        LISTSERV@WKUVX1.BITNET

The body of the letter should be this:

        SUBSCRIBE HARP-L "your real name"

The server should send you a help file and any letters already in the
archives (which is not many, YET!!).

If you have any questions or problems write me at:

        PIERCCM@WKUVX1.BITNET

If you know of others who would be interested, please let them know.


Thanks,

Chris Pierce
