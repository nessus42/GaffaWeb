Greetings Love-hounds;

   Since there was some talk about the different versions of Eat The Music 
lately,   I thought it might be nice to have a list of all the songs that have
different versions.  In compiling this list, I've purposely excluded bootlegged
live versions.  This list is not intended to show where one can find any
particular version.  Therefore I'm not going to list ALL the LPs, 12", 7", CD,
etc. where I can find the New Vocal version of WH for example.  But I will list
the most common place to find any particular version. 

   If I missed anything, please let me know so I can make updates to this list.

Peace

ron
