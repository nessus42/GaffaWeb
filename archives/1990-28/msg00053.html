CD Box Set In Canada

From: paullu@cs.ualberta.ca (Paul Lu)
Date: Fri, 26 Oct 90 23:01:11 GMT
Subject: CD Box Set In Canada
Newsgroups: rec.music.gaffa
Organization: University of Alberta, Edmonton, Canada
Sender: news@cs.ualberta.ca (News Administrator)

It appears there _will_ be a Canadian pressing of Kate's CD Box set.
This does not come from the record company but from four different
record stores that I have asked, all of whom expect to have the set in
stock within the next few weeks.  In fact, one store had October 26 as
the expected Canadian release date.

The price of the Canadian set is even more vague, although the store
that had Oct. 26 as the release date quoted me $150 CDN as the price.

The above information is guaranteed to be accurate with respect to
what I heard, but I do _NOT_ have any official information from the
record company.

I do not know if the Canadian set will be exactly the same as the UK
and Japanese releases, although I doubt very much if the CD tracks
will be any different.

American Love-Hounds may want to check with the A&A and HMV record
chains in Canada (who were among the record stores I visited) before
importing from the UK.  Caveat emptor.  (Or however that's spelled).

Eagerly awaiting it myself,
