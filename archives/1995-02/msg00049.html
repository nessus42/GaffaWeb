   If you have ftp access, a very good collection of Kate's (and a huge
number of other artist's) lyrics is available on:

ftp.uwp.edu

...in the directory...

/pub/music/lyrics/b/bush.kate

...the directory contains the following files...

dont_give_up
hounds.love
kick.inside
lionheart.
miscellaneous.
never.for.ever
red.shoes
sensual.world
the.dreaming
the.whole.story

   Uli's FAQ contains directions for using e-mail to retreive these
files if you don't have ftp. Also, access via WWW browsers seems
to be quicker and more likely to get a connection. 

   The original releases of all of Kate's recordings contained lyrics,
but EMI-America cheaped out on the "Nice Price" re-releases of her
music.
