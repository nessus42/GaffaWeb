This is a short listing from the index for the Lyrics Archives on
vacs.uwp.edu (131.210.1.1) These files are stored in the
/pub/lyrics/files directory.

A complete index can be found in the file:

/pub/lyrics/Index

If you know of a site containg Lyrics or have any lyrics files floating
around that you would like put in the archives, please mail
datta@vacs.uwp.edu.

Most of the files here were obtained from the Lyrics server which is no
longer in operation. If you have old files from the server that I am
missing, please forward them to me and I will add them to the archives
here.

	Dave datta@vacs.uwp.edu

The lyrics files are all available only in compressed form and they
currently have long file names, VERY slowly, I will be re-naming all of
the files to 8 character filenames. This will take quite a while as it
is not a priority.
