     When you get this error, the file you are getting is a directory, not a
file.  For instance, kb, is a directory.  To access the files in kb, type
"cd kb".  To see if files are normal or not, type "ls -l".  This will list the
names of the files in the current directory on the right.  If the file is a
directory, the far left character (column 1) will be a "d".  If it's an
actual text file or whatnot, it will be a "-".  If this isn't quite enough,
just email me.  (rec.music.gaffa.ftp  ;-)
