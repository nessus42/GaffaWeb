The Kate Bush VRML site is up and running at 

http:\\fly.hiwaay.net\~cbullard\index.htm.

Whew!!!  Lots of work...

You need a Netscape browser with a Live3D
plug in both of which you can get from the
netscape homepage (www.netscape.com)
for free! 

I've included links to the other kate pages
that I know about and that Wieland sent
to me (thanks, Wieland).  I will add more
if folks sent me the links.

There are four worlds there.  One is
a Kate-centered solar system with
some surprises.  Two are animations.
Be forwarned, the bitmaps take some
time to load across the network.  I'm
working on that problem.  the fourth
is a HomeGround Towers world 
dedicated to peter and krys.  at
some point, I'd like for this to become
a kate chat area.  this is a very big
file and if you load it, go get some
coffee.  it takes about 20 minutes
to load over the internet.
