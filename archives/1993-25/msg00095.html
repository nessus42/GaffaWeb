Just a quick note since someone contacted me by email with a problem
downloading and using the moments5.snd for Mac file I posted about
recently...

In order to get the flat unix file to copy correctly to the Mac, which
needs things divied up between resource and data forks, you need to
enable "MacBinary" mode in NCSA Telnet or Fetch.  In NCSA Telnet it's
under the File menu, and I think Fetch works it out automagically
unless you explicitly disable MacBinary.  This way the file should
come out ready for use... no ResEdit fiddling or anything.
