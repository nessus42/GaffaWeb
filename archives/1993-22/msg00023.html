Subject: Moments of Pleasure -- wav format

I have converted moments5.au to microsquish wave format (.wav)
and ftp'ed it to ftp.uu.net, /tmp (a previous post incorrectly
named this directory as /temp), file moments5.wav.

If you have multimedia sound on your windoze box, 
This file can be played with the MediaPlayer applet which is 
bundled with MS-Windoze 3.1.  It can also be played with any
number of windoze waveplayer utils available in the public
domain such as DROPWAVE or PLAYNOW.

A cautionary note: the file is 2.5MB (same as the .au).
