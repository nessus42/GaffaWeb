Someone recently expressed difficulty in getting to the Love-Hounds
archives.  They are available via anonymous FTP from the Internet host
"GAFFA.MIT.EDU".  The Internet number for this host is "18.62.0.11".
Once you have connected, look in the directory "/love-hounds".  In this
directory there is a file "lh-archive".  This file accumulates the
Love-Hounds mail as it comes in.  Eventually this stuff gets moved
into the directory "/love-hounds/archives".  In this directory, there
are files with names like 0034 and 0035.  The files are in
chronological order, and each file contains about 100 articles (except
the ones at the beginning, when I wasn't doing it that way yet).
