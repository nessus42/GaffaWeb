Just a short, gentle correction: it's The Single File. The main reason
I'm correcting this is to remind people...at the time The Single
File (box set) was released, Kate also released a videotape with the
same name, which has since been deleted (replaced by The Whole Story 
videos.) The Single File videotape was never released in the USA and 
Canada, and a few years ago a bootleg videotape surfaced bearing the
name *The Singles File*.

I've never heard of anyone bootlegging the singles box, but if the
name of the set you saw actually was The Single*s* File, then they're
bootlegs too. Otherwise, my correction isn't very important, but I
thought I'd throw it out anyway. The more common thing to happen to
the singles set was for (some) dealers to break open the set and sell
the singles individually. That's another reason why TSF box sets are
rare.
