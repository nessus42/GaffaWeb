rec.music.gaffa is really the gateway onto Usenet of an Internet
mailing list that existed before the newsgroup.  The mailing list is
called Love-Hounds, and if all the software is working right, the set
of messages received by people on the Love-Hounds mailing list should
be identical to the set of messages seen by people who read the
rec.music.gaffa newsgroup.  The reason that the newsgroup is not
called rec.music.love-hounds is because it was disallowed by the
facist netnews administrators.
