I've been a lurker on this list since late 87, but I've kept
everything. Six 720k floppies of digests stored in PKPAK format,
something like 8Mb of text.  I've also edited down the digests into
abridged versions containing only KT related messages - 3 floppies,
PKPAKed, about 4Mb.  (That gives Love-Hounds a 2:1 S+N:N ratio, not
bad.) As a separate archive file I've accumulated all of the
interviews. This file is about 350kb.

Most of these interviews were posted by IED.  Many thanks to him and
the others who took the time to transcribe them.

So, here's my offer: I think I can get away with uploading the
interview file and transmitting it directly to anyone on BITNET.  I
will have to transmit it as a binary file using the VM SENDFILE
command. You will have to download it to a PC in order to extract the
interviews. I can't make it available via FTP because I don't have a
publicly readable place to put it - maybe someone else can help with
that. I will also make any of these files available via snail-mail.
In any case, anyone who is interested should send me an email.

Note to |>oug: Do you have the archives going back to the beginning
of Love-Hounds?  If so, how can I obtain them?
