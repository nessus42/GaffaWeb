Subject: Box sets etc!!

For anybody interested there is a John Lennon box set to be released on the
first of October,it contains 73 tracks spread over four albums and a photo
booklet is included.
Regarding the box set wouldn't it have been a good idea to sell the b-sides etc
cd seperate,and make the box-set box available seperate  aswell.This would save
most fans the cost of paying out for cds they already have,and they could then
use the box to store the cds in!!This idea was used for the Beatles cd single
box set.
If you want to buy a good album,then buy 'And Love For All' by The Lilac Time!!
This has to be the best release of this year so far!!
