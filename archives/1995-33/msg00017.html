cbullard@HiWAAY.net (Len Bullard) Asks:

> I have been wondering
>if anyone has set out to create a VRML
>world for Kate Bush yet.  

Why, yes, I have!  If you go to TinyCRWU
(Telnet caisr2.caisr.cwru.edu 4201)
and, after all the obligitory intro stuff, type
@tel #61418

You will go to a little Kate MUSH area I set
up.

Now, here's the beauty of it:  TinyCRWU is
set up to work with Pueblo, a VRML browser
for MU*'s!  You can get YOUR copy at:

http://www.chaco.com 
(plug, plug)

[Note:  the rest of this is Nerd Talk.  If you're
a technophobe, feel free to skip the rest of
this letter]

I will SLOWLY be adding all the VRML goodies
(music, graphics, maybe some 3-D animation
if I'm up to it).  I say slowly because I can only
do it when I have time (which ain't often), AND
when I can use my housemate's Pentium (the
VRML stuff doesn't work on a 486, although it
still makes a good MU* client).  And I can take
away my housemate's keyboard when I pry it
from his cold, dead fingers.

Unfortunately, Pueblo doesn't QUITE work (yet)
on TinyCRWU - but that's a problem more with
TinyCRWU and how it handles SGML than with
Pueblo.  Luckily, I've been talking to Wizard, the
guy who WROTE TinyCRWU and, by an amazing
coincidence, one of the co-writers of Pueblo, and
he's going to fix that Real Soon Now.  When it's
all ready, I will definately announce it here, but until
then, it's still about as nice a Virtual Kate World
as we're going to get.

- Stev0 the Beta-tester
