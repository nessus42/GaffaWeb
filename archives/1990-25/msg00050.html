I seem to remember reading that the lyricserver has gone down...probably to
never reappear.  Seems some record company(s?) found out that the copyrighted
lyrics to lots o' music were available and got annoyed.  At least, that's
what I've heard.

I've used the server a bunch of times, and never had any problems...an
example might be:

LYRICS/ALBUM=NO BORDERS HERE

As you might suppose, this will get you the lyrics to all the songs on
the album "No Borders Here" by Jane Siberry.

This should be on its own line; there should be no subject.  You can
put multiple requests in a single message, however the server does cut replies
at a certain length, so don't request too much at once.  

On the header from one of the messages I got from the server, it lists
an address as "Lyrics%UMASS.BITNET@CUNYUM.CUNY.EDU"

As I've said, though, I think the lyricserver is down.
