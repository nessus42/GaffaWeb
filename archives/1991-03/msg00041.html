Tom Willett asks about getting some of the GIFs to display on his Mac.  
I happened to spend several hours in a Mac lab on campus last night working
this out for myself, so perhaps this will help...

First off, GIF files are binary.  Not MacBinary, but binary.  When you FTP
from gaffa.mit.edu (at least 'til our new pseudo-moderator gets set up),
make sure you set binary on FTP (by typing "binary [CR]".  When you download
to your mac, again, you need to use binary, but not MacBinary.

I was working solely by FTP last night, and don't know too much about
Mac telecomm software (I have a MacPlus emulator that works surprisingly
well; I had no problems displaying GIFs with Gif-Converter on it), but
the key is in making sure you use true binary and not macbinary.

That's really all there is to it...
