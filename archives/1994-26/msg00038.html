[Directory for ftp.uwp.edu

Note: In all the following directory listings, "..." should be replaced by
/pub/music/lists/gaffa/

Where you see a "d" in the first column, that means its a directory.
a "-" means its a file you can get.  The date shown is the date the
last time the file was modified.  If it shows a time instead of a year,
it was modified this year.  The number to the left of that is the size
of the file, and to the right of the date is the file name, of course.

All the comments here are shown as if you were to actually
ftp to this site.  All comments in brackets are mine.]

Welcome to the Love-Hounds/rec.music.gaffa archives.

The "kb" subdirectory contains GIFs, lyrics, and other miscellaneous
files.  The "archive" directory contains the archives of all messages
every sent to Love-Hounds.

All files in the archives have been compressed with the UNIX compress
command.  If you are on a system that lacks the uncompress command,
you can request the file without the ".Z" suffix and this FTP server will
automatically decompress the file before sending it.
