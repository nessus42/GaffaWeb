Here is the README from the ~ftp/katebush directory:

	Here are the Kate Bush images. All are in a very simple raster
format, as follows:
	3 VAX 32-bit longs:	first is either 1 or 3, for mono or color
				second is X, number of pixels per line
				third is Y, number of lines

	then the byte stream, 1 or 3 bytes per pixel.

	The program showimage, can read these images and dump them to a
SUN screen (8bit color). I have programs for the Ikonas, the Data
Translation DT2651, the Number Nine PC board, the QMS QUIC series laser
printers and Postscript bitmap laser printers, all of which understand
this format. (BTW showimage seems very inefficient -- don't blame me, I
didn't write it.)

	The SUN program is showimage.c + map.c
	The QMS program is qbit.c
	The Postscript program is psraster.c

The follow are 1:1 aspect ratio, from a slow-scan, high S/N CCD camera
