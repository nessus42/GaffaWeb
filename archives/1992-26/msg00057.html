I've been posting Tori Amos tour dates to the Really Deep Thoughts 
mailing list for a while now but it seems that there have been a lot
of requests for tour dates on other groups, as well.   Below is a list
of tour dates as I have them right now.  I will continue to post updated
lists to the RDT mailing list (rdt-request@gradient.cis.upenn.edu) but
you can obtain the latest list (updated every week or so) by sending
mail to "info@bsbbs.columbus.oh.us".  The contents of the message 
don't matter, just be sure you have a valid "From:" or "Reply-To:" 
header.
