>Just got the full sales prospectus on THE RED SHOES BOX SET and the RBG LITHO
>SET:
>
>KATE BUSH The Red Shoes Box Set
>
>This new box set series contains Kate Bush's full length CD "The Red Shoes",
>a
>pair of (UK) size 41/2 (as per Kate's feet) red satin Ballet slippers with
>ankle ribbon and a certificate of autheticity (sic) all in a red satin
>heart-shaped box.                 ^^^^^^^^^^^

   *Strange* usage of the word, no matter *how* it is spelled. This is
simply a box containing the normal release of the album, the one that
will be freely available in every record store, a pair of red ballet
slippers and a piece of paper. It has *no* connection with Kate, EMI
CBS, Novercia or anyone else. Kate makes the normal royalty on the
copy of _The Red Shoes_ and nothing more. This is simply a package
that *anyone* here could assemble. This has no collectable value that
I can see. A bootlegger's assurance that a package is a "limited
edition" is worth exactly nothing. Buy your *own* wooden box, run
to your local dancer's supply place and stick your own copy of _The
Red Shoes_ in it. It will have *more* value to you.  

>Limited Edition of 1,5000
