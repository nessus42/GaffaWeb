at least for me. One of the problems is that gryphon.com's internet feed
can't find the way to gryphon.com and gets stuck in a loop trying to find it.
Machine A says "hey, machine B, here's some data for gryphon.com", and
machine B answers immediately "hey, machine A, here's some data for
gryphon.com". Which causes machine A to say "hey, machine B, here's......"

The problem is known to bbnplanet.net, on whose machines the problem occurs.
They're investigating it currently. This problem definitely has to do with
the fact that gryphon.com seems to be unreachable with WWW or email (anyone
else gets warnings like "haven't been able to reach gryphon.com, will try for
a week" or something like that when sending email to gryphon.com?). It might
(but not has to) have to do with rec.music.gaffa being dead as well, because
gryphon.com normally sits directly next to the machine causing the problem
here, so maybe gryphon.com is just unable to get rid of the r.m.gaffa
articles.

I'll keep you posted this way if I find out something new.
