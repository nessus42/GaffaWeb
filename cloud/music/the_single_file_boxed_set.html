Cloudbusting -- Kate Bush In Her Own Words

The Single File Boxed Set

Released 23rd January 1984

I am so pleased at the reaction, both to the video and the boxed set of The Single File. It was a buzz for me to get the video released, and to see five years sitting in a little green box, but the feedback to them is stunning. Thanks to all of you for feeding-back. (1984, KBC 16)

Is it true that some of the singles in the single file boxed set have been re-mixed? And if so, which ones?

None of the singles have been re-mixed, but `` Sat In Your Lap'' and `` Breathing'' are the album mixes, and not as on the original singles. (1984, KBC 17)
